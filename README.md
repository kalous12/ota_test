# REDME

Changelog 格式如下：

```markdown
## VERSION

- Update time: YYYY-MM-DD
- Fixed: xxxx
- Added: xxxx
- Changed: xxxx
- Removed: xxxx
- Security: xxxx
```

- `Update time`：更新时间
- `Fixed`：修复的bug
- `Added`：新增的功能
- `Changed`：修改的功能
- `Removed`：删除的功能
- `Security`：安全相关的更新

例子：

```markdown
# changelog

## V0.7.0

- Update time: 2025-11-28
- Fixed: 修复了OTA升级过程中断电导致升级失败的问题
- Added: 增加了OTA升级过程中的进度显示
- Changed: 优化了OTA升级过程中的日志输出
- Removed: 移除了不再使用的OTA升级功能
- Security: 修复了OTA升级过程中的安全漏洞
```
